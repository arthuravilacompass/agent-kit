name: grill-me
description: Invoque quando o usuário pedir para "me grillar" / "grill me", pressionar uma decisão de design, ou antes de dar um plano por pronto para execução — entrevista estruturada e implacável que resolve dependências entre decisões uma a uma.
---

Interview me relentlessly about every aspect of this plan until we reach a shared understanding. Walk down each branch of the design tree, resolving dependencies between decisions one-by-one. For each question, provide your recommended answer.

Ask the questions one at a time.

If a question can be answered by exploring the codebase, explore the codebase instead.

Para interrogação estruturada com artefato formal (Gap Summary) no pipeline de tech breakdown, veja `core:spec-refine` (se disponível neste kit) em vez deste skill.
